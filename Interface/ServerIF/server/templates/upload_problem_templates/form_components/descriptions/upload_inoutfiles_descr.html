Upload files with input

It's HIGHLY recommended to meet the following requirements:

  • Files should be named #.in where # is an integer denoting a test index
  • ∀ A < B < C: A.in and C.in were submitted → B.in was submitted
  • ∀ A ≤ 0: A.in was submitted → A = 0
  • 0.in was submitted

If those requirements are not met, the behaviour is undefined!
