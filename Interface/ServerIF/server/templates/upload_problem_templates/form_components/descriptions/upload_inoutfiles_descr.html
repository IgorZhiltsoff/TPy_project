Upload files with in/output

It's HIGHLY recommended to meet the following requirements:

  • Files should be named #.in/out where # is an integer denoting a test index
  • ∀ A < B < C: A.in/out and C.in/out were submitted → B.in/out was submitted
  • ∀ A ≤ 0: A.in/out was submitted → A = 0
  • 0.in/out was submitted

If those requirements are not met, the behaviour is undefined!
