Upload {{semantics}} executable

It's HIGHLY recommended to meet the following requirements:
  • The {{semantics}} should be an executable! If it is written in, say, Python3, it should have a shebang at its top. The interpreters are guaranteed to be stored in /bin, thus all the shebangs should match #!/bin/interpreter. Supported interpreters include:
    • python3
    • sh, bash, zsh
    • perl
    If you want to use some exotic interpreted language not listed above which just does not support compilation, you may still TRY to submit it and hope that an according interpreter is present among the {{bin_dir_file_count}} files in our /bin directory
  • When running, {{semantics}} should not take up more than:
    • {{time_limit}} seconds
    • {{memory_limit_megabytes}} * 106 bytes (megabytes)

Upon violation of the points above, the system will terminate testing and return a special type of verdict
